Пример 3: animation-direction, animation-iteration-count

Анимација со употреба на дополнителни два елементи окрај основните: animation-direction, animation-iteration-count.

  • animation-direction: animation-direction: normal | reverse | alternate | alternate-reverse | initial | inherit;
    • Специфира во која насока да се одвива анимацијата (нормално, наназад, напред-назад)
  • animation-iteration-count: animation-iteration-count: number | infinite| initial | inherit;
    • Специфицира колку пати да се повтори анимацијата (или да се иизвршува бесконечно).

Опис: Логото се накосува хоризонтално за 30 степени, потоа се зголемува 2 пати по x-оска. Анимацијата со animation-iteration-count се повторува 2 пати и се извршува во обратен редослед поради animation-direction: reverse.

Употребени стилови и трансформации:
  • skewX(angle): Накосување за некој агол
  • scaleX(number): Зголемување по x-оската
HTML/CSS код

HTML 
<div class="animationContainer">
    <div class="primer3">
        <img src="images/css_logo.png" />

CSS 
.animationContainer:hover > .primer3 {
    animation-name: primer3;
    animation-duration: var(--anim-duration);
    animation-iteration-count: 2;
    animation-direction: reverse;
}

@keyframes primer3 {
    40% {
        transform: skewX(30deg);
    }
    50% {
        transform:scaleX(2);        
    }
    100% {
        transform: translateX(0);
    }
}

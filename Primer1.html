Пример 1: keyframes, animation-name, animation-duration

Анимација со употреба на трите основни елементи: keyframes, animation-name, animation-duration.

  • @keyframes: @keyframes animation-name {keyframes-selector {css-styles;}}
    • animation-name: Го содржи називот на анимацијата и е задолжително
    • keyframes-selector: Ги дефинира чекорите на анимацијата во проценти (од 0% до 100%)
    • css-styles: Едно или повеќе применливи CSS својства или трансформации
  • animation-name: animation-name: keyframename | none | initial | inherit;
    • Го специфицира името за поврзување со соодветниот @keyframes
  • animation-duration: animation-duration: time | initial | inherit;
    • Го специфицира времетраењето на анимацијата за еден цел циклус

Опис: Објектот најпрво се движи хоризонтално до крајот на рамката, потоа ротира околу својата вертикална оска, па се враќа во почетната состојба. Времетраењето на анимацијата е 8 секунди.

Употребени стилови и трансформации:
  • translateX(x): Транслација долж x-оската
  • rotateY(angle): Ротација околу у-оската
HTML/CSS код

HTML 
<div class="animationContainer">
    <div class="primer1">
        <img src="images/css_logo.png" />
    </div>
</div>
CSS 
.animationContainer:hover > .primer1 {
    animation-name: Primer1;
    animation-duration: var(--anim-duration);
}

@keyframes Primer1 {
    0% {
        transform: translateX(0);
    }
    40% {
        transform: translateX(calc(var(--anim-width) 
                              - var(--img-width))) rotateY(0);
    }
    50% {
        transform: translateX(calc(var(--anim-width) 
                              - var(--img-width))) rotateY(180deg);
    }
    60% {
        transform: translateX(calc(var(--anim-width) 
                              - var(--img-width))) rotateY(0);
    }
    100% {
        transform: translateX(0);
    }
}

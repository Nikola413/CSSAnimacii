Пример 2: animation-delay, animation-fill-mode

Анимација со употреба на дополнителни два елементи окрај основните: animation-delay, animation-fill-mode.

  • animation-delay: animation-delay: time | initial | inherit;
    • Го специфицира времето на доцнење на анимацијата
  • animation-fill-mode: animation-fill-mode: none | forwards | backwards | both | initial | inherit;
    • Специфицира дали и кога да се задржи некое својство на анимацијата (пред или по завршување на анимацијата).

Опис: Логото се движи хоризонтално, потоа ја менува бојата на позадината и се враќа во првобитната положба ротирајќи околу својата хоризонтална оска. Анимацијата има доцнење од 3s и на крај со animation-fill-mode ја задржува бојата на позадината.

Употребени стилови и трансформации:
  • translateX(x): Транслација долж x-оската
  • rotateX(angle): Ротација околу x-оската
HTML/CSS код

HTML 
<div class="animationContainer">
    <div class="primer2">
        <img src="images/css_logo.png" />
    </div>
</div>
CSS 
.animationContainer:hover > .primer2 {
    animation-name: primer2;
    animation-duration: var(--anim-duration);
    animation-delay: 3s;
    animation-fill-mode: forwards;
}
                                            
@keyframes primer2 {
    0% {
        transform: translateX(0);
    }
    40% {
        transform:translateX(calc(var(--anim-width)
                             - var(--img-width))) rotateX(0);
    }
    50% {
        transform: translateX(calc(var(--anim-width)
                              - var(--img-width))) rotateX(0);
    }
    100% {
        transform: translateX(0) rotateX(360deg);
        background-color: #672312;
    }
}

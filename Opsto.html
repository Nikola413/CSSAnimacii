CSS Анимации

CSS Анимации се ефекти кои овозможуваат да се анимира даден елемент со промена на едно или повеќе својства на елементот. За нив е доволна употребата на HTML и CSS и нема потреба од дополнителен код (на пр. Java Script).

За да се изработи една функционална CSS анимација, потребно е дефинирање на:

  1. HTML елемент што ќе се анимира.
  2. CSS правила за поврзување на анимацијата со елементот, со задолжително дефинирање на својствата: animation-name и animation-duration.
  3. Група од @keyframes правила - интервали во кои се дефинираат клучните моменти за почеток и крај на движењата.

Анимациите можат да се прилагодуваат согласно потребите со примена на некои следните својства:

  • animation-delay: Одлагањето на стартот на анимацијата
  • animation-direction: Насоката на движење на елементот
  • animation-duration: Времето за кое анимацијата изведува еден циклус
  • animation-fill-mode: Стилот на елементот кога анимацијата завршува или пред да започне
  • animation-iteration-count: Бројот на повторувања на анимацијата
  • animation-name: Називот на еден @keyframe од анимацијата
  • animation-play-state: Дали анимацијата да се извршува или да се паузира
  • animation-timing-function: Кривата на брзина (забрзување) на анимацијата

Сите овие наброени својства можат да бидат дефинирани во рамки на својството animation, заради заштеда на простор во css датотеката. Значи, сите овие претставуваат под-својства на animation.
